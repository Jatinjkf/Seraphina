**Common Issues:**

**Q: Seraphina isn't responding?**
- Check if the bot is online
- Ensure you're using slash commands (`/`)
- Wait a moment and try again

**Q: Item name has duplicates?**
- Seraphina auto-numbers them: `item #1`, `item #2`
- Use `/rename` to make names unique

**Q: Can't see partner's items?**
- Both must accept the partnership
- Use `/view-partner` to verify

**Q: Reminders not coming?**
- Check your Discord notification settings
- Ensure DMs are enabled for the server
